# To-do-app-challenge
A web-based app I took as a challenge from my mentor 
Project: TaskFlow — A Focus-Driven To-Do App chick
Overview
TaskFlow is a web-based to-do application built to demonstrate front-end competence using HTML, CSS, and JavaScript.
The app focuses on clarity, intentional task management, and visual feedback without overwhelming the user.
This project exists to prove skill, structure, and thoughtful UI logic, not just task creation.
Core Features (Mentor Requirements Covered)

1. Task Creation (Pop-Up Only)
   A task can only be created by clicking “Add Task”. Clicking the button opens a modal / pop-up form
   Task inputs include:
   -Title (required)
   -Description (Optional)
   -Due date
   -Priority (Low / Medium / High)
   -A cancel and create task button.

2. Task properties
   -Title
   -Due date
   -Priorities    

3. Task interaction
   Clicking a task opens a pop app menu to
   -Edit details
   -Mark as complete
   -Delete task.

4. Task Categories (Automatic)
   Tasks are automatically grouped into:
   -Pending
   -Completed
   -Overdue (based on due date logic)

5. All Tasks View
   A unified “All Tasks” view that shows:
   -Pending
   -Completed
   -Overdue
   Visual indicators distinguish each state clearly.

6. Progress Tracking
   A main progress bar that shows:
   -Total tasks
   -Completed tasks
   -Pending tasks
   Updates dynamically as tasks are completed.

7. Sidebar Navigation
   Left sidebar contains:
   -App name
   -A mini progress bar under the app name
   -Navigation items (All Tasks, Pending, Completed, Overdue)
   “New Project” section at the bottom (static for MVP)

8. Search Functionality
  -A search bar to filter tasks by title
  -Works across all task states.

9. Light & Dark Mode
   -Toggle between light and dark themes
   -Theme preference persists during the session.

10. Badge System
    Clicking the badge icon opens a badge panel.
    Badge panel displays:
    -Total tasks completed
    -Milestone badges (e.g. “Task master”, “Legendary”)
